# Parstagram

Instagram clone using a Parse backend.

Created For: CodePath - iOS Development

## Part 1

### User Stories

#### Required

- [x] (1pt) User sees app icon in home screen and styled launch screen.
- [x] (1pt) User can sign up to create a new account.
- [x] (1pt) User can log in. 
- [x] (3pts) User can take a photo, add a caption, and post it to the server. 
- [x] (4pts) User can view the last 20 posts. 

#### Bonus

- [x] (1pt) User can pull to refresh. 
- [x] (2pts) User can load past posts infinitely. 

## Part 2

### User Stories

#### Required

- [x] (1pt) User stays logged in across restarts.
- [x] (1pt) User can log out.
- [x] (3pts) User can view comments on a post.
- [x] (5pts) User can add a new comment.

#### Bonus

- [x] (2pts) User can add a profile picture.
- [x] (2pts) Profile pictures are shown for posts and comments.
